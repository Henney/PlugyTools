# PlugyTools
A program to read PlugY stashes

# Building
mvn clean package

# Running
The jar is executable, you can run this program with:

java -jar plugytools.jar "path\to\\_LOD_SharedStashSave.sss"

Or with:

plugytools.jar "path\to\\_LOD_SharedStashSave.sss"
